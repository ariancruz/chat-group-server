## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger
#### IP:PORT/swagger

## Environment Variable
 
PORT = 3000 by default

MONGODB_URI = Url to mongo server

TOKEN_SECRET = Secret key for encrypt the JWT

GEMINI_API_KEY = Gemini secret key

GEMINI_MODEL = gemini-2.0-flash-exp by default
